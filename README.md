# awsx

`awsx` is a CLI tool for retrieving temporary credentials via AWS SSO.

## Installation

### Homebrew
```
brew tap brewdex/awsx
brew install awsx
```

### Binary Download
Get the binary from the [releases page](https://github.com/gerdou/awsx/releases).

### Build from Source
With Go installed, run:
```
go install github.com/gerdou/awsx@latest
```
